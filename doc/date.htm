Marking Output With Current Date

Syntax
Description
Example

Syntax
$date

Description
Displays the date corresponding to the current execution of OMhelp in the output.

Example
This command is useful for identifying the version of the web site corresponding to this execution of OMhelp. For example, the input 
 
     The help system you are reading was compiled by OMhelp
     on $date.
     Please use this date,
     and the date printed out when OMhelp executes,
     to identify the version corresponding to
     any bug reports or questions about OMhelp.
results in:
The help system you are reading was compiled by OMhelp on Oct 12, 2019. Please use this date, and the date printed out when OMhelp executes, to identify the version corresponding to any bug reports or questions about OMhelp.
Input File: omh/date.omh
