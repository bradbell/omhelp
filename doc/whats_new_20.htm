Description
This section contains a list of changes to OMhelp during 2020 (in reverse order by date). Its purpose is to assist you in learning about changes between versions.

01-31
  1. Add detection and error message when a srcthisfile start pattern matches the start pattern itself (instead of skipping to another copy of the pattern).
  2. Remove the verbatim, srcfile, srccode wish list item because srcthisfile contains the important features that were missing.


01-30
Add the srcthisfile command.

01-29
The error message when a requested version of a start pattern can not be found has been improved. To be specific, the version of the patter is reported; see skip .
