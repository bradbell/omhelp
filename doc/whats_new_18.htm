Description
This section contains a list of changes to OMhelp during 2018 (in reverse order by date). Its purpose is to assist you in learning about changes between versions.

11-16
The option to use a _jump_table for the current section of the documentation was added.

11-15
Improve documentation for navigate command. In addition, in its example 
 
     Down_up_2%Down_up_2%
was corrected to 
 
     Down_up_2%_up_2%

10-31
Add a image item to the wish list.

05-13
Upper case letters versus lower case letters no longer matters when determining the language corresponding to a file extension for srccode and srcfile .

03-13
The font used for sub-headings was changed to be same size as headings but normal instead of bold font; see example subheading .

03-11
The install download omhelp procedure was changed to use the following command to determine the version number: 
 
     grep '^SET(omhelp_version' CMakeLists.txt
