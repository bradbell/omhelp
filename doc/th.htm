Indexing A Value

Syntax
Description
Example

Syntax
$th index$$

Description
This command is used for displaying an index in front of the letters th. The text index does not contain any $ characters. In addition, leading and trailing white space in index is ignored.

Example
For example, 
 
     Element $th (i,j)$$ of $latex X$$ is denoted by $latex X_{i,j}$$
results in the following output
     Element (i,j)-th of @(@ X @)@ is denoted by @(@ X_{i,j} @)@
