Using the Title of Another Section

Syntax
See Also
Description
Example

Syntax
$title tag$$
See Also
tref

Description
The argument tag specifies the cross reference tag for the another section. The title of the section corresponding to tag is placed in the text where this command appears.

Example
The following input 
 
     $title section$$: $mref/section/$$
produces the following output
     Specify The Title For A Section: section
Note that section is the tag in this example.
