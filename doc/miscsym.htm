Miscellaneous Latex Symbols

Syntax
Description
     Arrow Symbols

Syntax
$latex Command $$

Description
Display in Latex mode the miscellaneous symbol specified by Command where Command is one of the Latex commands in the tables below. The following tables contain the miscellaneous symbols in Table 3.7 of Lamport

The MathML standard displays a diamondsuit as black while Lamport's book displays it as white.

Note that \Box and \triangle are in the list of symbols that MS Internet Explorer does not display properly.
