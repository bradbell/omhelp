Cross Referencing By Displaying a Section's Title

Syntax
$tref tag$$
See Also
title , cref

Description
The argument tag specifies the cross reference tag for the link. The title of the section corresponding to tag is used to reference the link in the current section. Leading and trailing white space in tag is ignored.
