Displaying an Accent Over a Vowel

Syntax
See Also
Description
Example
Reference

Syntax
$'v
See Also
dollar , rmark , dollar , cmark , newlinech

Description
Displays the character v with a accent over it where v is a vowel; i.e.,
      Á, É, Í, Ó, Ú, Ý, á, é, í, ó, ú, ý
There can be no space or other characters between the ' and the character v .

Example
The input 
 
     The word fossil in English has the same meaning as
     f$'osil in Spanish.
results in
     The word fossil in English has the same meaning as fósil in Spanish.

Reference
The spell checker does not currently recognize this command. For example, in the example above the word sil must be added to the spelling exceptions using the spell command.
