     Relation Symbols

Syntax
$latex Command $$

Description
Display in Latex mode the relation symbol specified by Command where Command is one of the Latex commands in the tables below. The following tables contain the relation symbols in Table 3.5 of Lamport (except that \Join is not included and generates an error message).

Note that \preceq, \models, and \bowtie are in the list of symbols that MS Internet Explorer does not display properly. Also note that \preceq is in the list of symbols that Firefox does not display properly.
