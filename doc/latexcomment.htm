Latex Comment Command

Syntax
Description
Example

Syntax
$latex % text $$

Description
All of the character in text are ignored where text is any sequence of characters not including a new line character or the double dollar $$.

Example
There are three Latex comment commands in the following input 
 
$latex \[
     % align columns: first right, second centered, and third left
      \begin{array}{rcl}
          a b & =      & c   \\ % first equation
              & \vdots &     \\
          x   & =      & y z    % last equation
     \end{array}
\] $$
The Latex mode output below corresponds to the input above: @[@ % align columns: first right, second centered, and third left \begin{array}{rcl} a b & = & c \\ % first equation & \vdots & \\ x & = & y z % last equation \end{array} @]@
