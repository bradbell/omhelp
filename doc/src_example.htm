Contents of src_example.py
Note that a newline is automatically added at the beginning of a file: 
     #! /usr/bin/python3
     import sys
     # BEGIN USAGE
     usage = 'src_example.py value'
     # END USAGE
     if len(sys.argv) != 2 :
          sys.exit(usage)
     value  = float( sys.argv[1] )
     square = value * value
     output = 'value * value = ' + str(square)
     print(output)
