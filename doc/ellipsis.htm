Latex Ellipsis Commands

Syntax
Description

Syntax
$latex command$$

Description
Display in Latex mode the specified ellipsis where command is one of the following:

command   $latex 1, 2, command, N$$
 \cdots   @(@ 1, 2, \cdots, N% @)@
\ddots   @(@ 1, 2, \ddots, N% @)@
\ldots   @(@ 1, 2, \ldots, N% @)@
\vdots   @(@ 1, 2, \vdots, N% @)@
