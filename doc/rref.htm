Generate One Row of a Jump Table

Syntax
$rref tag$$
See Also
mref , tref , children

Description
The argument tag specifies the cross reference tag for the row in the jump table. This command creates one row in a jump table. The tag is hypertext linked and forms the first column in the row. The title of the section corresponding to tag is placed in the second column for the corresponding row. Leading and trailing white space in tag is ignored.

Example
The following input: 
 
     $table
     $rref mref$$
     $rref rref$$
     $rref tref$$
     $rref cref$$
     $tend
produces the following output:
mref Multiple Tag Based Cross References
rref Generate One Row of a Jump Table
tref Cross Referencing By Displaying a Section's Title
cref General Cross Reference Within a Web Site
Input File: rref.omh
