Do Not Break Text Into Separate Lines

Syntax
$nobreak text$$
See Also
pre , math

Description
All of the characters in text are displayed on the same line. There can be no $ characters in text . (Note that the pre command can be used to ensure that text appears on the same line but the pre command changes to a pre-formatted font .)

Example
The input 
 
     The pone numbers are:
     $nobreak 206 555 1212$$,
     $nobreak 207 555 1212$$,
     $nobreak 208 555 1212$$,
     and
     $nobreak 209 555 1212$$.
would create the following output
      The pone numbers are: 206 555 1212 , 207 555 1212 , 208 555 1212 , and 209 555 1212 .
Input File: nobreak.omh
