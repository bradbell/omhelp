Page Layout and Text Formatting

Introduction
The commands listed below control the placement of text and the fonts used to display text.

Command Limited
The effect of the following commands is limited to the command itself:
nobreak Do Not Break Text Into Separate Lines
pre Preformatted Text With a Uniformly Spaced Font
codep Preformatted Code Text
codei Switch Between Code and Italic Preformatted Font
color Setting Colors
hilite Automatic Highlighting and Cross Referencing Certain Tokens
th Indexing A Value

Not Command Limited
The effect of the following commands may extend across other commands:
center Centering Text
table Specifying a Table and Its Rows and Columns
list Commands Used to Display Lists of Items
special Special Characters in Omhelp Input
font Specifying Fonts Type and Size
Input File: format.omh
