Latex Stacking One Thing Above Another

Syntax
Description
Example

Syntax
$latex \stackrel{%above%}{%commands%}$$

Description
Display in Latex mode the output corresponding to commands with the specified output above it.

Example
The input 
 
     $$latex A \stackrel{a'}{\rightarrow} B
               \stackrel{b'}{\rightarrow} C
     $$
generates the following output
      @(@ A \stackrel{a'}{\rightarrow} B \stackrel{b'}{\rightarrow} C @)@
Input File: latex/stackrel.omh
