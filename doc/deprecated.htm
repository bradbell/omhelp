OMhelp Commands that Have been Deprecated

Purpose
-l, Deprecated 2006-09-12
-i, Deprecated 2006-09-12

Purpose
The following correspond to valid OMhelp commands that are discouraged from use. They are included below mainly so that one can understand the intention of old OMhelp input files that use theses commands
wspace Processing a Special Input Character as White Space
xref Specifying a General Cross Reference With in Web Site
fend Dividing A Section Into Frames
syntax Specifying Command or Function Syntax
math Math Mode
number Displaying Simple Numbers In Math Mode

-l, Deprecated 2006-09-12
The OMhelp command line argument -l has been deprecated. It has been replaced by (and has the same effect as) the -image_link command line argument.

-i, Deprecated 2006-09-12
The OMhelp command line argument -i has been deprecated. It has been replaced by (and has the same effect as) the -image_file command line argument.
